# Futures-Sim Historical Data Directory

> 外部历史行情数据存放目录。
> 只存 CSV 原始文件，不进行任何转换或加工。
> 数据入口规范见 `schema.md`。

---

| 文件 | 说明 | 来源 |

**无真实历史数据。** 当前所有策略验证结果仅适用于流程验证，不代表任何市场真实表现。

---

## 如何添加数据

1. 获取原始历史数据（手工下载 / 第三方导出 / 后续 provider 模块）
2. 转换为 `schema.md` 定义的 CSV 格式
3. 放入此目录
4. 通过 replay/converter.py 加载

数据来源和准确性由导入者负责。核心引擎不校验数据真实性。

## 命名规范

```
<品种>_<年份>.<扩展名>
```

示例：
```
RB_2025.csv        # 螺纹钢 2025 年数据
CU_2024-2025.csv   # 沪铜 2024-2025 年数据
```

## 约束

- ❌ 不直接从交易所 API 写入此目录
- ❌ 不自动下载
- ❌ 不缓存实时行情
- ✅ 仅接受已落盘的 CSV 文件
